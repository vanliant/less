- #### vsCode中设置easy-less

  - ##### setting.json中设置

    ```javascript
    "less.compile": {
            "compress": false, // true => remove surplus whitespace
            "sourceMap": false, // true => generate source maps (.css.map files)
            "out": "../css/", // false => DON'T output .css files (overridable per-file, see below)
    }
    ```
